REQUIREMENTS
============

- py3-matplotlib:       (optional) Plotting support.

- graphviz imagemagick: (optional) Graph generation.
- py3-cryptography:     (optional) IPsec, WEP, PKI and TLS support.
- sox:                  (optional) VOIP support.
